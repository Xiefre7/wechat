# 导题斩题小工具 - 管理端网页

## 目录结构

```
管理端网页/
├── index.html          # 主页面（SPA，含登录 + 全部管理功能）
├── css/
│   └── style.css       # 样式表
├── js/
│   ├── auth.js         # 认证模块（Token 管理）
│   ├── api.js          # API 通信模块
│   └── app.js          # 主应用逻辑（路由、视图、交互）
└── README.md
```

## 部署步骤

### 1. 部署云函数

1. 在微信开发者工具中打开此项目
2. 右键 `cloudfunctions/admin-api` → 上传并部署：云端安装依赖
3. 在云开发控制台 → 云函数 → admin-api → 创建 HTTP 触发器
4. 复制 HTTP 触发器 URL

### 2. 配置 API 地址

编辑 `js/api.js`，将 `BASE_URL` 替换为实际的云函数 HTTP 地址：

```javascript
const BASE_URL = 'https://your-env-id.service.tcloudbase.com/admin-api'
```

### 3. 部署网页

将 `管理端网页` 文件夹部署到任意静态托管服务（如微信云开发静态托管、Nginx 等）。

### 4. 首次使用

- 访问网页，使用默认账号登录：
  - 用户名：`admin`
  - 密码：`admin123`
- **请登录后立即修改密码**

## 功能模块

| 模块 | 功能 |
|------|------|
| 📊 数据概览 | 题库数、题目数、资讯数、待处理反馈等统计 |
| 📚 题库管理 | 创建/编辑/删除官方题库，管理知识点层级 |
| 🎯 真题管理 | 创建/编辑/删除历届真题库 |
| 📝 题目管理 | 逐题录入、编辑、批量文本导入、搜索筛选 |
| 📰 资讯管理 | 发布/编辑/删除考试资讯（纯文本） |
| 📮 反馈查看 | 查看用户反馈、标记已处理 |

## 技术说明

- **前端**: 纯 HTML/CSS/JS，零框架依赖，打开即用
- **后端**: 微信云开发 HTTP 云函数 (`admin-api`)
- **数据库**: 微信云数据库（复用小程序数据）
- **认证**: Token 会话机制，24 小时过期
